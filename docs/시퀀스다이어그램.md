## 시퀀스 다이어그램

#### 유저 대기열 토큰 발급 API
```mermaid
sequenceDiagram
    actor 사용자
    participant api
    participant 회원
    participant 대기열

    alt 대기열 토큰 없을 때
        사용자 ->> api: 토큰 발급 요청
        api ->> 회원: 사용자 검증 요청
        회원 -->> api: 사용자 검증 결과 반환
        alt 사용자 검증 성공
            api ->> 대기열: 토큰 생성 요청
            대기열 -->> api: 생성된 토큰과 대기 순서 및 잔여 시간 정보 반환
            api -->> 사용자: 대기 순서 및 잔여 시간 정보 반환
        else 사용자 검증 실패
            api -->> 사용자: 사용자 검증 실패 메시지 반환
        end
    else 대기열 토큰 있을 때
        api ->> 회원: 사용자 검증 요청
        회원 -->> api: 사용자 검증 결과 반환
        alt 사용자 검증 성공
            api ->> 대기열: 유효한 토큰 조회 요청
            alt 토큰 유효성 검사
                대기열 -->> api: 유효한 토큰과 대기 순서 및 잔여 시간 정보 반환
                api -->> 사용자: 대기 순서 및 잔여 시간 정보 반환
            else 토큰 만료 처리
                토큰 -->> api: 만료된 토큰 반환
                api -->> 사용자: 만료된 토큰에 대한 오류 메시지 반환
            end
        else 사용자 검증 실패
            api -->> 사용자: 사용자 검증 실패 메시지 반환
        end
    end

```

#### 좌석 예약 요청 API
```mermaid
sequenceDiagram
    actor 사용자
    participant api
    participant 대기열
    participant 콘서트
    participant 좌석
    participant 결제

    사용자 ->> api: 콘서트 좌석 예약 요청
    api ->> 대기열: 유효성 검증 요청
    alt 유효성 검증 성공
        대기열 -->> api: 유효성 검증 성공
        api ->> 콘서트: 콘서트 일정 및 좌석 예약 가능 여부 조회 요청
        콘서트 -->> api: 좌석 예약 가능 여부 반환
        alt 좌석 예약 가능
            api ->> 좌석: 좌석 임시 배정 요청 (5분간 유지)
            좌석 -->> api: 임시 배정 완료
            api -->> 사용자: 좌석 임시 배정 성공 정보 반환 (결제 필요)
            사용자 ->> 결제: 결제 요청
            alt 결제 성공
                결제 -->> api: 결제 성공
                api ->> 좌석: 좌석 최종 배정 요청
                좌석 -->> api: 좌석 최종 배정 완료
                api -->> 사용자: 예약 및 결제 성공 메시지 반환
            else 결제 실패
                결제 -->> api: 결제 실패
                api ->> 좌석: 좌석 임시 배정 해제 요청
                좌석 -->> api: 임시 배정 해제 완료
                api -->> 사용자: 결제 실패 및 좌석 해제 메시지 반환
            end
        else 좌석 예약 불가능
            api -->> 사용자: 좌석 예약 불가능 메시지 반환
        end
    else 유효성 검증 실패
        대기열 -->> api: 유효성 검증 실패
        api -->> 사용자: 유효성 검증 실패 메시지 반환
    end
```

#### 결제 API
```mermaid
sequenceDiagram
    actor 사용자
    participant api
    participant 회원
    participant 결제
    participant 대기열
    participant 좌석

    사용자 ->> api: 결제 요청
    api ->> 회원: 사용자 검증 요청
    회원 -->> api: 사용자 검증 결과 반환
    alt 사용자 검증 성공
        api ->> 결제: 결제 정보 처리 요청
        결제 -->> api: 결제 처리 결과 반환
        alt 결제 성공
            api ->> 좌석: 좌석 소유권 배정 요청
            좌석 -->> api: 소유권 배정 결과 반환
            api ->> 대기열: 대기열 토큰 만료 요청
            대기열 -->> api: 토큰 만료 결과 반환
            api -->> 사용자: 결제 성공 및 좌석 소유권 배정 정보 반환
        else 결제 실패
            api -->> 사용자: 결제 실패 메시지 반환
        end
    else 사용자 검증 실패
        api -->> 사용자: 사용자 검증 실패 메시지 반환
    end
    
```
